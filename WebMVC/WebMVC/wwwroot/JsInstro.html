JavaScript 特性展示

JavaScript 主要特性

  1. 直譯/即時編譯（JIT），在瀏覽器中執行
  2. 操作 DOM（Document Object Model）
  3. functions可以當變數、參數、回傳值
  4. 動態型別（Dynamic Typing），同一個變數可以裝不同型別的值，但也因此更容易踩到型別轉換的坑。
  5. 弱型別（Weak Typing），像 1 + "2" => "12"；"5" - 1 => 4，（加法偏向字串串接，其他運算多半轉成數字）

JavaScript 變數宣告

  1. let，區塊作用域（block scope），可重新指派
  2. const，常數，區塊作用域（block scope）
  3. var，不推薦使用，生命週期混亂，不易控制，早期JS的用法

JavaScript 型別總覽

  1. string：字串
  2. number：數字（包含整數/浮點數，NaN 也是 number）
  3. bigint：大整數（123n）
  4. boolean：布林
  5. undefined：未賦值
  6. null：表示「空」

JavaScript 的比較運算 == vs ===

  1. ===：不做型別轉換（推薦）
  2. ==：會做型別轉換（容易出意外）

✅ 1. 瀏覽器中執行

JavaScript 是「前端語言」，可以直接在瀏覽器裡跑

✅ 2. 操作 DOM

JavaScript 可以選取網頁上的元素，並動態修改，DOM（Document Object Model）是 JS 連接 HTML 的橋樑

✅ 3. 事件導向

JavaScript 可以偵測使用者的操作，常見事件：click, mouseover, keydown, submit，事件是前端互動的靈魂

✅ 4. 動態型別

JavaScript 變數不需宣告型別，型別會在執行時期判斷，好處：彈性高、語法簡潔，壞處：容易出錯（目前前端框架多數支援 TypeScript ）

✅ 5. 弱型別

不同型別可以「自動轉換」，但有時會產生意料之外的結果
