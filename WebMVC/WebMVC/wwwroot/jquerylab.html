互動式學習 jQuery 的核心功能

📖 什麼是 jQuery？

jQuery 是一個快速、輕量且功能豐富的 JavaScript 函式庫，由 John Resig 於 2006 年建立。 它的設計理念是 "Write Less, Do More"（寫更少，做更多），大幅簡化了 JavaScript 程式設計。

✨ jQuery 的核心優勢：

  • 簡潔的語法：使用 $ 符號和 CSS 選擇器，讓 DOM 操作變得直觀易懂
  • 跨瀏覽器相容：自動處理不同瀏覽器的差異，無需擔心相容性問題
  • 強大的 DOM 操作：輕鬆選取、修改、新增或刪除 HTML 元素
  • 豐富的事件處理：簡化使用者互動事件的綁定與管理
  • 動畫效果：內建淡入淡出、滑動等動畫效果
  • AJAX 支援：簡化非同步 HTTP 請求的撰寫
  • 插件生態系：擁有豐富的第三方插件可擴充功能

🎯 為什麼要學習 jQuery？

雖然現代前端開發有 React、Vue 等框架，但 jQuery 仍然廣泛應用於：

  • 快速開發小型專案或原型
  • 維護既有的舊專案（許多網站仍在使用）
  • 學習 JavaScript DOM 操作的基礎概念
  • 需要輕量級解決方案的場景

💡 jQuery vs 原生 JavaScript 比較

// 原生 JavaScript
document.getElementById("demo").innerHTML = "Hello";
document.querySelectorAll(".item").forEach(el => el.style.color = "red");

// jQuery - 更簡潔！
$("#demo").html("Hello");
$(".item").css("color", "red");

2️⃣ CSS 類別操作

🔍 addClass vs toggleClass 的差異：

addClass - 只會「新增」CSS 類別。如果元素已經有該類別，不會有任何變化。適合用在確定要加上某個樣式的情況。

toggleClass - 「切換」CSS 類別。如果元素沒有該類別就新增它，如果已有該類別就移除它。適合需要開關效果的場景（像燈的開關）。
