CSS Box Model 盒⼦模型 Box Model 盒⼦模型是 CSS 的基礎，CSS 的佈局主要基於「Box 模型」，Box 模型主要是由Content (內容)、Padding (內邊距)、Border (邊框) 和 Margin (外邊距)四個部分所組成，其 中Padding、Border 和 Margin 都採⽤相同的語法定義寬度。
1. 4個邊分開定義，上、右、下、左
2. 2個值，上下、左右
3. 3個值，上、左右、下
4. 1個值，同時4個邊相同定義
CSS Box Model 盒⼦模型的實際 UI 佔用寬度取決於 box-sizing 屬性值，該屬性值有content-box／border-box　二種
小技巧是一律加上 box-sizing: border-box;，避免尺寸失控
(預設值)content-box：width/height 只包含內容區域

在 content-box 模式下，width 和 height 指的是內容區域的寬高。 padding 和 border 會額外加到設定的寬高上。
設定寬度：200px;
實際寬度=250px（200px + 40px padding + 10px border）

border-box：width/height 包含 padding 和 border

在 border-box 模式下，width 和 height 包含了 padding 和 border。 實際的內容區域會自動縮小以容納 padding 和 border。
設定寬度：200px;
實際寬度=200px（包含 padding 和 border）

  1. 避免意外的溢出問題
  2. 更容易預測元素的實際大小
  3. 簡化寬度計算
