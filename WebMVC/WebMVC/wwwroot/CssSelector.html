屬性選擇器-根據 HTML 屬性進行選取

偽類選擇器（hover）- 元素在某些狀態/條件下的樣式，我要用不同樣式。

  • :hover → 滑鼠移上去時
  • :focus → 輸入框被選到時
  • :active → 按下去那瞬間
  • :disabled → 被禁用的表單
  • :checked → checkbox/radio 被勾選
滑過我會變色

偽元素選擇器（::first-line）- 元素的虛擬部分（非 DOM 中實際存在的元素）

  • ::before / ::after → 在內容前/後「插入」一個可樣式化的東西
  • ::first-letter → 第一個字母
  • ::first-line → 第一行
  • ::selection → 被反白選取的部分
