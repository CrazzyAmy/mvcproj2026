HTML CSS display

display 每個HTML元素都有⼀個預設的display值，但不同的元素會有不同的display預設值。⽽⼤部 分元素的預設值通常是 block（區塊元素） 或 inline（⾏內元素），因此display將會影響元素在版⾯上所 佔⽤的區域，例如div 就是⼀個標準的區塊元素(display預設值為block)。

1. display: block

這是 block 元素

block 特性： - 獨占一行 - 可以設定寬高 - 可以設定 margin 和 padding

2. display: inline

這是 inline 元素
inline 特性： - 不會獨占一行 - 無法設定寬高 - 只能設定水平方向的 margin 和 padding
